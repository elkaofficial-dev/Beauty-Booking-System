> Полностью асинхронная система записи в салон красоты с Telegram ботом и админ-панелью

## ✨ Особенности

- 🤖 **Telegram Bot** — интуитивный интерфейс для клиентов
  - Выбор услуг из каталога (маникюр, стрижка, массаж)
  - Быстрый выбор доступных дат
  - Ручной ввод даты в нужном формате
  - Валидация дат и проверка доступности

- 📊 **Admin Web Panel** — управление записями
  - Просмотр всех активных записей в одной таблице
  - Удаление записей одним кликом
  - Чистый и минималистичный UI
  - Работает на FastAPI

- ⚡ **Полностью асинхронная архитектура**
  - asyncio/asyncpg для быстрой работы с БД
  - Масштабируемость под нагрузку

- 🔐 **Надежное хранилище данных**
  - PostgreSQL для предотвращения конфликтов
  - Автоматическое создание таблиц при первом запуске

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- PostgreSQL 12+
- pip/poetry для управления пакетами

### 1️⃣ Клонируем репозиторий
```bash
git clone https://github.com/yourusername/beauty-booking.git
cd beauty-booking
```

### 2️⃣ Создаём виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate  # Windows
```

### 3️⃣ Устанавливаем зависимости
```bash
pip install -r requirements.txt
```

### 4️⃣ Настраиваем переменные окружения
```bash
cp .env.example .env
```

Заполняем `.env` файл:
```env
BOT_TOKEN=тут_ваш_bot_token_от_botfather
DATABASE_URL=postgresql://user:password@localhost:5432/beauty_shop
```

### 5️⃣ Запускаем бота и админ-панель

**В отдельных терминалах:**

```bash
# Терминал 1: Запуск бота
python bot_main.py

# Терминал 2: Запуск админ-панели
python admin.py
```

✅ Готово! 
- Телеграм бот работает и слушает обновления
- Админ-панель доступна на `http://localhost:5000`

## 📖 Использование

### 👥 Для клиентов (Telegram)

1. Найдите бота в Telegram
2. Нажмите `/start`
3. Выберите нужную услугу (💅 Маникюр, ✂️ Стрижка, 💆‍♀️ Массаж)
4. Выберите удобную дату или введите свою
5. Готово, вы записаны! ✅

### 🔧 Для администратора (Веб-панель)

1. Откройте `http://localhost:5000`
2. Видите таблицу со всеми записями
3. Хотите удалить запись? Кликните кнопку "Удалить"
4. Запись тут же исчезнет из системы

## 🛠️ Структура проекта

```
beauty-booking/
├── bot_main.py          # Основной файл Telegram бота
├── admin.py             # Админ-панель на FastAPI
├── requirements.txt     # Зависимости проекта
├── .env.example         # Пример переменных окружения
└── README.md           # Этот файл
```

### 📄 bot_main.py
- **Классы**: `Book` (FSM для диалога)
- **Функции**:
  - `is_free(d)` — проверка свободности даты
  - `kb_services()` — клавиатура выбора услуг
  - `kb_dates()` — клавиатура выбора дат
  - Обработчики сообщений и callback запросов

### 📄 admin.py
- REST API на FastAPI
- GET `/` — вывод всех записей в HTML таблице
- POST `/del` — удаление записи по дате
- Встроенный HTML с Jinja2 шаблонам

## 🚨 Обработка ошибок

### ❌ Неверный формат даты
Если пользователь введет дату в неверном формате — бот попросит повторить в виде `ДД.ММ.ГГГГ`

### ❌ Дата уже прошла
Система не позволит выбрать дату в прошлом

### ❌ Дата занята
Если на дату уже есть запись, система покажет свободные даты
